# Customer Segmentation Project

This project performs customer segmentation using Machine Learning.

## Objective
To group customers based on annual income and spending score using K-Means Clustering.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Mall Customers Dataset

## Machine Learning Algorithm
K-Means Clustering

## Features
- Data analysis
- Customer grouping
- Elbow method visualization
- Cluster visualization

## Output
The project segments customers into different groups based on purchasing behavior and income.
